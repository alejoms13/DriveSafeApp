# DriveSafe 🚗💨

**DriveSafe** es una aplicación móvil diseñada para mejorar la seguridad vial mediante la prevención de distracciones al volante. La app ayuda a los usuarios a mantenerse enfocados en la carretera, reduciendo el riesgo de accidentes causados por el uso del smartphone mientras se conduce.

## 🚀 Características Principales

- **Prevención de Distracciones:** Gestiona inteligentemente las notificaciones y alertas para mantener la vista en el camino.
- **Modo Conducción:** Detecta automáticamente cuándo estás conduciendo y ajusta el comportamiento del teléfono.
- **Análisis de Seguridad:** Proporciona información sobre los hábitos de conducción para promover un comportamiento más seguro.
- **Integración de Emergencia:** Acceso rápido a servicios de emergencia en caso de ser necesario.

## 🛠️ Stack Tecnológico

- **Plataforma:** Android
- **Lenguaje:** Kotlin
- **Sistema de Construcción:** Gradle (Kotlin DSL)
- **Backend:** Firebase (Autenticación, Firestore, etc.)

## 📦 Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/alejoms13/DriveSafe.git
   ```
2. Abre el proyecto en **Android Studio**.
3. Sincroniza Gradle y construye el proyecto.
4. Ejecuta en un emulador o dispositivo físico.
